# Contributing to Calorie Tracker

Thank you for your interest in contributing to Calorie Tracker! We welcome contributions from everyone.

## How to Contribute

1.  **Report Bugs**: If you find a bug, please open an issue describing the problem, steps to reproduce, and any relevant logs or screenshots.
2.  **Suggest Features**: Have an idea to make the app better? Open an issue with the "feature request" tag.
3.  **Submit Pull Requests**:
    *   Fork the repository.
    *   Create a new branch for your feature or bug fix (`git checkout -b feature/awesome-feature`).
    *   Commit your changes with clear, descriptive messages.
    *   Push to your fork and submit a pull request.

## Code Style

*   We follow standard Java coding conventions.
*   Ensure your code is well-commented and easy to understand.
*   Check for any lint warnings before submitting.
